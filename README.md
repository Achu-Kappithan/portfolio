### 💻 Local Development

1. **Clone the repo:**
   ```bash
   git clone https://github.com/Achu-Kappithan/portfolio.git
   ```
2. **Setup:**
   ```bash
   cd portfolio/portfolio
   npm install
   ```
3. **Run:**
   ```bash
   npm start
   ```
